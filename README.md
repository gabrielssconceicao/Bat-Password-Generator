## Building

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
git clone https://github.com/gabrielssconceicao/Bat-Password-Generator.git
$ cd react-native-bat-pass-generator
$ npm install
$ npm run start
```
